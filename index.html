Get packages that install and run the same way, every time

Ωpm only has all-JS packages with verified builds. We don't run any scripts before or after install. We check that the packages are still the same at runtime, every time.

Sign your work

Use GPG and git to sign and verify authorship of every package.

Keep a secure line open

Developers use the same GPG key to send critical updates and security messages to users.
